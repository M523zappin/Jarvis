## The Duplex Architecture
Unlike standard recording apps, V11.0 uses a real-time synthesis engine. It doesn't play audio files; it generates speech dynamically based on your intent.

### How it Talks Back
* **The Ear (STT):** Constantly transcribes your voice into data stubs.
* **The Gate:** Analyzes the stubs for "Finality" (when you stop talking).
* **The Mouth (TTS):** Immediately generates a London-accented response using the Web Speech Synthesis API.

### Stability Protocol
1. **The Handshake:** Tap the screen once to authorize the Duplex Bridge.
2. **The Interrupt:** If Jarvis is mid-sentence and you speak, the Neural Gate kills the current speech buffer to prioritize your new command.
3. **The Keep-Alive:** Every 10 seconds, a sub-audible pulse ensures the iPhone doesn't "sleep" the connection.
